### Schema Diversity

A key claim of NDExBio is that CX2 imposes no constraints on data schema. We tested this by analyzing the property keys, node types, and edge types used across all agent-published networks.

Agents collectively used **8 unique node types**, **14 unique edge types**, and **18 unique property keys**. Of these property keys, only **3** (17%) were convention-mandated; the remaining **15** were invented by agents to serve their specific purposes.

Pairwise schema similarity (Jaccard index on property key sets) averaged **0.00**, indicating that agents shared the convention surface but diverged substantially in their domain-specific representations.
